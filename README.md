# Blog Management System

## 📌 Project Overview
A simple Blog Management System developed using PHP and MySQL. This project allows users to register, log in, and manage blog posts with full CRUD functionality.

## 🚀 Features
- User Registration & Login
- Dashboard
- Add Blog Post
- View Blog Posts
- Search Blog Posts
- Edit Blog Post
- Delete Blog Post
- Logout
- Responsive UI using Bootstrap

## 🛠 Technologies Used
- PHP
- MySQL
- HTML
- CSS
- Bootstrap 5
- Git & GitHub
